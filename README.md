# Depuis

  Track time elapsed since important dates.

  Built with Expo (React Native Web). Supports French and English, dark/light mode.
  # ⏳ Depuis — Gardez le compte de vos moments précieux

  [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
  [![Platform](https://img.shields.io/badge/platform-Android-brightgreen.svg)](#)

  **Depuis** est une application mobile élégante et minimaliste conçue pour suivre le temps écoulé depuis vos événements les plus importants. Qu'il s'agisse de célébrer une victoire personnelle (comme l'arrêt du tabac) ou de se souvenir d'une date spéciale, Depuis transforme vos jalons en une expérience visuelle apaisante.

  ---

  ## ✨ Fonctionnalités

  * **Timers Personnalisés :** Créez autant de compteurs que vous le souhaitez avec des titres explicites.
  * **Design Pastel & Moderne :** Une interface douce (Neumorphism / 3D Soft) pour une navigation sans stress.
  * **Suivi Précis :** Visualisez instantanément les jours et les heures écoulés depuis chaque événement.
  * **Mode Hors Connexion :** Vos données restent accessibles même sans réseau avec une page d'erreur dédiée et stylisée.
  * **Organisation Intuitive :** Gérez vos timers d'un simple coup d'œil.
